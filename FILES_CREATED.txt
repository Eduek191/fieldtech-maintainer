API ENDPOINTS
────────────────────────────────────────────────────────────────────────────

Equipment Management:
  GET    /api/equipment              - Retrieve all equipment
  GET    /api/equipment/:id          - Get single equipment
  POST   /api/equipment              - Create new equipment
  PUT    /api/equipment/:id          - Update equipment
  DELETE /api/equipment/:id          - Delete equipment

Maintenance Logging:
  GET    /api/logs                   - Get all maintenance logs
  GET    /api/logs/equipment/:id     - Get logs for equipment
  POST   /api/logs                   - Create maintenance log
  DELETE /api/logs/:id               - Delete log

Schedule Management:
  GET    /api/schedules              - Get all schedules
  GET    /api/schedules/upcoming     - Get upcoming maintenance (14 days)

Reports & Export:
  GET    /api/reports/monthly        - Monthly overview report
  GET    /api/reports/equipment-status - Equipment detail report
  GET    /api/reports/technician-activity - Technician statistics
  POST   /api/reports/export         - Export data to JSON

System:
  GET    /api/health                 - API health check
